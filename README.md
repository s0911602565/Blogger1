# 專案名稱blogger
此為簡易的部落格平台，使用SpringBoot進行專案設計

## 專案介紹
此專案為簡易的Bloger，包含前台流程系統，基於SpringBoot與Hibernate實現。
- 部落格系統
  - 首頁
  - 登入
  - 註冊
  - 開帖
  - 瀏覽
  - 回帖
  - 會員中心
- 後台管理系統
  - 點擊次數
  - 回復次數
  - 排程寫入redis
  

 
## 使用到的技術
| 技術                | 說明       |
|-------------------|----------| 
| SpringBoot        | Web應用開發  | 
| Hibernate	        | ORM框架    | 
| MySQL	            | 資料庫      | 
| Spring Validation | 資料驗證     | 
| Quartz            | 任務排程     | 
| Css               | 前端框架     | 
| VueJs             | 前端框架     | 
| Redis             | 記憶體資料存放區 |
